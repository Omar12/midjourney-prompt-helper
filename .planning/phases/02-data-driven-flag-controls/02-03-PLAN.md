Serializer flag-emission algorithm (replace lines 31-35):
1. Build flagParts: string[] = []
2. If draft.selectedVersionId is not null, call getVersionParameter(draft.selectedVersionId);
   if it returns a non-null string, push to flagParts
3. Loop over draft.flags (already ordered and version-filtered by App.tsx before passing in):
   call serializeFlag(flagId, value); if non-null, push to flagParts
4. Build descriptors = parts.join(', ')
5. Return: if flagParts.length === 0, return descriptors; else if descriptors is truthy,
   return descriptors + ' ' + flagParts.join(' '); else return flagParts.join(' ')
