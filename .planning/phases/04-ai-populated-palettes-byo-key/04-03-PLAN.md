<objective>
## Phase Goal

**As an** image creator, **I want to** enter my intent and my own LLM API key to trigger AI-suggested palette options, **so that** I can assemble a complete Midjourney prompt faster than writing it by hand.

**This plan:** First user-visible slice. After this plan, the user can open a settings modal, enter their API key (masked), save it, and clear it — the key stored locally, never sent to a first-party server. Simultaneously, the palette session state store is ready for Plan 04-04 to wire the suggest flow into.

Purpose: Deliver the key management UX (KEY-01/02/03) that gates the suggest feature, and establish the palette state shape (AI-03/04) so the suggest flow and failure UX can be implemented in one final plan.

Output: paletteSession Zustand store + tests; SettingsModal + KeyInput components.
</objective>
