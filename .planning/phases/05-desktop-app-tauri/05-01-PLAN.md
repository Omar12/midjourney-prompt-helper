**As a** macOS user, **I want to** launch Midjourney Prompt Helper as a native desktop app built from the same codebase, **so that** I can build, save, and recall prompts locally without a browser.

Scaffold the Tauri 2 wrapper alongside the existing Vite SPA without touching `src/`. </objective>
